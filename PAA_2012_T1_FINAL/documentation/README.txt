Passos para executar o programa.
1 - Existem dois arquivos .jar que são os executaveis e uma pasta "teste" que contém os arquivos de entrada.
2 - Para adicionar novos arquivos, basta copia-los para dentro da pasta, os arquivos devem ter a extensão .txt ou .dat.
3 - Para executar os arquivos use os seguintes comandos:
	java -Xmx1024m -jar pph.jar > pph.log
	java -Xmx1024m -jar frascos.jar > frascos.log
